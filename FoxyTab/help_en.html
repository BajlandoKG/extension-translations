Preamble

Actions works from any Tab, active or background.
Firefox restricts some functions (e.g. copy/promt/warning) on some URLs. FoxyTab v2.11+ works on all URLs.
Firefox is changing in preparation for FF57 and there might be unforeseen issues between FF53-57 due to API changes.

How to Use

Tab Counter

Toolbar icon with a badge showing the number of opened tabs in all windows. Due to some issues beyond FoxyTab's control, the number can become inaccurate. Clicking the Toolbar icon will recount the tabs. To preserve resources, Tab Counter will start counting as soon as a new tab is opened or closed. If desired, you can move the Toolbar icon via Customize Toolbar.

Duplicate Tab

Duplicate tab next to the Tab and make it active

Close Duplicate Tabs

Close all duplicate tabs, ignoring location.hash

Close Tabs to the Left

Similar to the built-in "Close Tab to the Right", close all Tabs to the left of current tab, warns if there are more than 1 tab to close.
New in v2.11: FoxyTab no longer closes the Pinned Tabs, unless Ctrl is pressed.

Merge All Windows

Merge all tabs from all windows into the current window. To preserve security, when merging to/from a Private window, tabs are closed in target window and the same URLs \re opened in the current window. When moving tabs between Private and Normal windows for security reasons, in Firefox, privileged URLs are not moved (e.g. chrome: URLs, javascript: URLs, data: URLs, file: URLs, privileged about: URLs (for example, about:config, about:addons, about:debugging). In other words, when moving tabs between Private and Normal windows, only tabs with URL starting with HTTP/HTTPS/FTP will be moved.

Close Other Windows

Close all other opened windows.

Save Tab as PDF

Save as PDF is not available on Mac OS X or before Firefox 56.
I may add some user preferences later.

    Keep all the tabs from the same host, close others

    Move all Tabs to the Left/Right to a new Private Window

    Reloads the Tab accordingly

    Stop the Automatic Reload

Options

Please note that changes will not take effect until they are saved.

Tab Counter

Enable or Disable the Tab Counter, and/or change the Badge Background Color.
Note: Disabling the Tab Counter will also disable its process and listeners.

Custom Reload

Select a Custom Reload value (set to between 6 to 30 minutes at the moment)

Date

Add multiple Date displays in preferred languages in the Toolbar popup.
Note: The selected dates will be set as toolbar icon title and can be seen on mouse-over.

Clock

Add multiple Clock displays in preferred languages and Time-zones in the Toolbar popup.
Note: Not all city names are added (there are 400+ cities). Let me know if you want any to be added.

Context Menu

Enable or Disable each item on the context-menu and/or re-order the items. Moving the main menu will move its sub-menus.

  • Disabling the Tab IP also disables its process and listeners as well as Country Flag Display of host location (v2.10) (IPv4 at the moment). Flag Display also shows if the page was loaded from cache.
  • Moving a parent menu also moves its sub-menus
  • Disabling a parent menu also disables its sub-menus
  • After disabling menus, it may look better to disable the associated separators

Restore Defaults

Restores the Defaults settings (good in case of error). Click "Save" to apply changes.

Theme

Create/Import/Export/Apply your own Lightweight Themes (Firefox 55+)
Lightweight themes are made up of a "header" graphic image file, which skins the default Firefox UI background.

Creating a Theme Header Image
The header image is displayed as the background of the top of the browser window, nestling in behind the toolbars, address bar, search bar and the tab strip. It will be anchored to the top-right corner of the browser window.

Image Requirements

  • Dimensions should be 3000px wide × 200px high
  • PNG, JPG, APNG, SVG, GIF (animated GIF isn’t supported) or Data:URI file format
  • Image must be no larger than 300 KB in file size

Tips

  • Subtle, soft contrast images and gradients work best; highly detailed images will compete with the browser UI.
  • Firefox may reveal more of the lower portion of the image if another toolbar or other UI element is added to the top of the window.
  • The upper right-hand side of the image should have the most important information—as a user increases the width of the browser window, the browser reveals more of the left-hand side of the image.
  • Ultrawide monitors can run to 3440px wide, to look good on these monitors, either increase the size of the image or ensure the left of the image fades to a solid colour.

Import

Import ready FoxyTab Themes (JSON files), or import an image into the theme that you are making.

Imported Image will be converted to Data:URI entered in "Header Image" section, if it is empty and if not, in the "Additional Backgrounds" section.

FoxyTab Theme manifests can be imported into FoxyTab in the following JSON format, using any available properties listed on MDN theme.
The main difference is that images must be in Data:URI format. Maximum size allowed for the JSON is set to 500kb.

The minimum requirements for a theme are "accentcolor" & "textcolor".

{
  "name": "Theme Name",
  "description": "A short explanation of your theme",
  "version": "1.0",
  "homepage_url": "https://.../",
  "author": "your name",

  "theme": {
     "images": {
       "headerURL": "data:image/jpg;base64 ...",
       "additional_backgrounds": [ "data:image/png;base64 ..."]
     },
     "properties": {
       "additional_backgrounds_alignment": [ "top" ],
       "additional_backgrounds_tiling": [ "repeat"  ]
     },
     "colors": {
       "accentcolor": "green",
       "textcolor": "#000"
     }
   }
}

Export

Export the theme as JSON to archive or share.

Editing

  • Save: Save theme to storage
  • Update: Update the saved theme
  • Delete Delete saved theme
  • Apply: Apply the currently displayed theme. Any changes will also be applied. You can Apply a theme without even saving it.
  • Reset: Reset the current theme (Firefox 56+)
    In Firefox 55, you can reset the theme by going to the browser Themes page in about:addons and selecting another theme
